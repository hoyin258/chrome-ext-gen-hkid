Privacy Policy for HKID Generator

Last updated: 2024

This privacy policy governs your use of the "HKID Generator" Chrome Extension.

1. Data Collection The Extension does NOT collect, store, or transmit any personal information, browsing history, or user data to any external servers.

2. Local Storage The Extension uses the storage permission solely to allow users to save optional local notes alongside generated IDs. These notes are stored locally on your device and are not accessible by the developer or any third parties.

3. Remote Code The Extension does not use any remotely hosted code. All logic is performed locally within the extension package.

4. Third-party Access No data is shared with third parties because no data is collected.

5. Contact If you have any questions, please contact the developer through the support link in the Chrome Web Store.
